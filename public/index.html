API Reference

Powerful endpoints for every use case. From natural language to structured JSON, SolForge has you covered.

POST /api/build
Build transactions from structured intent objects with precise control over parameters.
POST /api/build/natural
Transform natural language descriptions into executable Solana transactions.
POST /api/build/multi
Combine multiple operations into one atomic transaction for complex workflows.
POST /api/decode
Decode and analyze any Solana transaction to understand its structure and purpose.
POST /api/estimate
Estimate transaction costs including fees, rent, and compute units before building.
GET /api/protocols
List all supported protocols and their current status and capabilities.
GET /api/protocols/:name/schema
Auto-discover required parameters and options for any specific protocol.
GET /api/docs
Access complete API documentation with examples and integration guides.
