API Reference

Powerful endpoints for every use case. From natural language to structured JSON, SolForge has you covered.

POST /api/build/natural

Transform natural language descriptions into executable Solana transactions with intelligent protocol detection.

POST /api/build

Build transactions from structured intent objects with precise control over parameters and execution flow.

POST /api/build/multi

Combine multiple operations into one atomic transaction for complex workflows and batch operations.

POST /api/decode

Decode and analyze any Solana transaction to understand its structure, purpose, and estimated impact.

POST /api/estimate

Estimate transaction costs including fees, rent, and compute units before building or executing.

GET /api/resolve

Resolve token addresses, trading pairs, and protocol information using DexScreener integration.

GET /api/protocols

List all supported protocols with their current status, capabilities, and supported operations.

GET /api/docs

Access complete API documentation with examples, integration guides, and best practices.
